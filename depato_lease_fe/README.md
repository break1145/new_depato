## 前端启动流程

1. 打开gitbash，使用以下命令克隆前端代码

   ```bash
   git clone https://github.com/break1145/depato_lease_fe.git
   cd depato_lease_fe
   ```

2. ```bash
   npm install
   npm run dev
   ```

3. 访问`http://localhost:5173/`

环境依赖：node，npm。需要预先安装node环境并切换npm镜像或设置代理
